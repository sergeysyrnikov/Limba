# Limba

### Сервер на DRF для приложения лимба.

Последовательность запуска:
* Установить docker на компьютер
* Запустить комманду для создания образов `docker-compose build` из каталога где расположен файл docker-compose.yml.
* Запустить контейнеры `docker-compose up -d` из каталога где расположен файл docker-compose.yml.
